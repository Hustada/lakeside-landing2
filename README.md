# Lakeside Landing

Welcome to Lakeside Landing, a serene private lake retreat in rural Nebraska. This repository contains the source code for our booking website.

 [Visit the Live Site](https://lakesidelanding.co)

<img src="public/readmescreenshot.png" alt="Lakeside Landing Screenshot" width="100%">

## About

Lakeside Landing offers a unique escape on an 80-acre private property featuring:
- A serene private lake for fishing and boating
- Cozy cabin accommodating up to 8 guests
- Modern amenities in a rustic setting
- Abundant wildlife and outdoor activities

## Features

- Elegant, responsive design with vintage aesthetic
- Interactive photo gallery showcasing the property
- Real-time booking calendar integration
- Smooth animations and transitions
- Mobile-optimized experience
- Contact form with engaging animations
- Local wildlife information
- Detailed amenity listings

## Tech Stack

- Next.js 14
- TypeScript
- Material-UI (MUI)
- Framer Motion for animations
- Lucide Icons
- Responsive design principles

## Development

To run the development server:

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
